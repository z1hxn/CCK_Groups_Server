# 1. 각 역할 정보

### 선수 (출전)

- 대회에서 기록을 재는 선수이다
- 시작하면 큐브제출대에 큐브를 제출하고 대기석에 착석한다
- 러너의 안내에 따라 스테이션으로 이동한다
- 심판의 안내에 따라 스테이션에서 기록을 잰다
- 기록 측정이 끝나면 **혼자** 다시 대기석으로 이동한다
- 모든 기록 측정(5회)가 끝나면 혼자 경기장 밖으로 나간다

### 심판

- 선수를 관리하는 역할이다
- 스테이션(자리) 에 상시 착석해있는다
- 러너가 데려오는 선수를 1:1로 관리한다
- 선수의 기록 측정을 안내하고 감독한다
- 측정이 끝나면 기록을 확정하고 선수를 대기석으로 보낸다

### 러너

- 선수와 큐브의 이동을 담당한다
- 착석하지 않고, 계속 경기장을 돌아다닌다
- 큐브제출대의 큐브들을 가리개에 넣고, 스크램블석으로 보낸다
- 스크램블석에서 섞인 큐브를 가지고, 대기석에서 선수를 호명한 후, 심판 스테이션으로 안내한다
- 측정이 끝난 맞춰진 큐브를 심판에게서 받고, 다시 스크램블석으로 보낸다
- 이 과정들을 한 자리에 멈춰서 기다리지 말고, 계속 경기장을 돌아다니며 유동적으로 각 절차가 필요할 때에 수행한다. 즉 위의 작업들은 시간 순서대로 해야 하는 것이 아니며, 순서가 섞일 수 있다.

### 스크램블러

- 큐브를 섞는 일을 한다
- 러너가 가져다주는 큐브를 섞고, 이를 테이블에 내려놓으면 러너가 이를 가져간다.

# 2. 대회 진행 방식

1. 선수가 큐브를 큐브 제출대에 제출한다.
2. 러너가 이를 수집하여 스크램블석에 전달한다
3. 스크램블러가 큐브를 섞는다
4. 돌아다니던 러너가 섞인 큐브를 집는다
5. 큐브 가리개 안에 큐브와 ID카드가 넣어져 있는데, 이를 보고 러너가 선수를 호명한다
6. 선수는 러너의 호명을 듣고 일어난다
7. 러너는 선수를 데리고, 빈 자리로 이동한다
8. 심판에게 큐브가 들어있는 큐브 가리개를 전달하고, 선수를 그 자리에 앉힌다.
9. 심판은 큐브를 놓고, 선수가 준비되었다 하면 가리개를 연다
10. 선수는 준비되면 큐브를 솔빙하고, 맞춘다
11. 심판이 큐브 확인 후 기록을 입력하고, 모두 확인되면 선수를 다시 대기석으로 보낸다 (이때 선수는 혼자 돌아간다)
12. 심판 혼자스테이션에 남은 상황에서, 심판이 큐브 가리개에 맞춰진 큐브와 ID카드를 넣고 손을 들면, 러너가 이를 회수하여 스크램블석으로 전달한다


# 3. 각 인원수

아래는 각 **조 별로** 몇 명이 배정되는지를 설명한다
이는 자동 생성 전 관리자가 인원수를 입력할 것이기 때문에 직접 하지는 않아도 되나, 모든 배정에 있어서 배정 원리를 알고 참고한 채로 작업을 수행하라

- 선수 : 기록을 측정하는 출전자로, 신청을 통해 받았기 때문에 출전 인원이 고정적이며, 예외사항이 아닌 이상 변동되지 않는다. 전체 종목별 인원을 그 라운드의 조 개수로 나눈 후, 소수점이 없고 최대한 수가 비슷하도록 조정하여 확정한다
- 심판 : 각 스테이션을 관리하는 감독자러, 선수를 1:1로 감독한다. 그러나 선수 인원과 동일하게 배정하는 것이 아닌, 스테이션 기반으로 배정되기 때문에 (이동석 방식의 경우) 심판의 수는 출전자보다 적고, 러너가 계속해서 선수들을 바꾸어 준다. 따라서 기다리는 선수들이 생긴다
- 러너 : 큐브와 선수의 이동을 관리하는 이동자로, 각 조별로 2~4명 정도 배정한다.
- 스크램블러 : 큐브를 섞는 사람으로 각 조별로 1~3명 정도 배정한다.

# 4. 배정 로직

### 기본 로직
- 아래는 사이트상에서 자동으로 배정되는 기능이므로 특히 더 주의하여 학습하도록 한다
- 조직위원은 심판과 러너를 배정하지 않는다.
- 선수, 심판, 러너, 스크램블러를 배정할 때에는, 반드시 해당 조에 하나씩만 배정한다.
- 기본적으로 조직위원 제외 모든 선수가 심판과 러너 역할 후보자이며, 각 조에서 연속으로도 스태프를 할 수 있다.
- 배정은 아래 순서로 한다.
    1. 각 조에 출전하는 선수를 먼저 배정한다
    2. 스크램블러 배정 방법에 따라 선수와 겹치지 않게 배정한다
    3. 선수, 스크램블러랑 겹치지 않게 러너를 배정한다
    4. 선수, 스크램블러, 러너와 겹치지 않게 심판을 배정한다

### 선수 배정

- 기본적으로, 선수 배정은 이미 설정한 조들의 인원수에 맞추어 완전히 랜덤으로 배정한다.
- 하지만, 사용자 Role이 ADMIN인 조직위원의 경우, 대회를 운영해야 하기 때문에, 최대한 다른 조로 분산시킨다. (auth.cubingclub.com/api/auth/info/{{cckId}} 의 position 값으로 확인가능)

### 심판 배정

- 설정한 조의 심판 인원수에 맞추어 랜덤으로 배정한다

### 러너 배정

- 설정한 조의 러너 인원수에 맞추어 랜덤으로 배정한다

### 스크램블러 배정

- 스크램블러는 큐브를 잘못 섞으면 안되므로, 큐브를 잘하는 사람에게 배정한다. 따라서 조편성 자동화 전에 스크램블러를 할 수 있는 선수들을 미리 관리자가 고르게 한다.

# 5. 현재 구현되어있는 기능의 활용법

- 조 설정에서 사용하는 API를 통해 각 조별 심판, 러너, 스크램블러, 출전자 인원수를 미리 확인한다. 혹시 이 API가 없을경우 구현한다.
- 자동 조편성은 해당 대회에 존재하는 모든 조편성을 일단 초기화한후 처음부터 진행한다.


# 6. 페이지 구현 방식

- 자동 조편성 메뉴에 들어가면 먼저 참가하는 조직위원 목록이 나오고, 스크램블러 할 사람을 고르게 된다.
- 고른 후 자동 배정하기를 누르면, 모든 조편성 초기화랑 같은방법으로 초기화에 동의하게 한다.
- 자동배정이 끝나면 안내를 띄운다.
